# GIF Storyboard — Day 01

## Should this topic use a GIF?

Yes, optional. Excel navigation is easier to explain with motion.

## GIF idea

Title: From Messy Workbook to Clean Analytics Workbook

Duration: 6–8 seconds  
Format: 1080 × 1080 square GIF or short MP4

## Frames

1. Frame 1:
   Messy workbook with tabs named Sheet1, Sheet2, Final, Copy.
   Text: "Messy workbook = slow analysis"

2. Frame 2:
   Tabs start renaming.
   Sheet1 → Stores
   Sheet2 → Products
   Final → Sales_Orders
   Copy → Inventory_Audit

3. Frame 3:
   Headers freeze at the top.
   Text: "Freeze headers"

4. Frame 4:
   Filters appear on each column.
   Text: "Filter-ready data"

5. Frame 5:
   Clean structure shown:
   Stores | Products | Sales_Orders | Inventory_Audit | README

6. Frame 6:
   Final message:
   "Clean navigation comes before advanced analysis"
   Footer:
   💼 Vinay Kumar Kalluri · 🐙 DataWithVinay

## When to post GIF instead of image

Use GIF when teaching:
- navigation
- workflow steps
- before/after cleanup
- pipeline movement
- dashboard interaction

Use static image when teaching:
- definitions
- formulas
- cheat sheets
- interview traps
